## plain html with ES modules

```sh
open nanocal.html
open ranger.html
```

## with frameworks

make sure to update `jsx` in `poi.config.js`

```sh
npm install
npm run vue:nanocal
# or
npm run vue:ranger
# or
npm run react:nanocal
# or
npm run react:ranger
```
